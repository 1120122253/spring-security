Spring Security is a framework that provides <<servlet-authentication,Authentication>>, <<servlet-authorization,Authorization>>, and <<servlet-exploits,Protection Against Exploits>>.
With first class support web applications deployed in a <<servlet,servlet container>> and <<reactive,reactive applications>>, it is the de-facto standard for securing Java applications.
